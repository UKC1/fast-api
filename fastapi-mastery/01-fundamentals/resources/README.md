# 01-Fundamentals 학습 자료

## 🎯 자료의 목적
빠른 참조와 복습을 위한 핵심 정보 모음입니다. 실습 중 막힐 때나 개념 정리가 필요할 때 활용하세요.

## 📋 예정 자료 목록

### 치트시트 (Cheat Sheets)
- `async_cheatsheet.md`: 비동기 프로그래밍 핵심 문법 모음
- `fastapi_patterns.md`: 자주 사용되는 FastAPI 패턴들
- `performance_checklist.md`: 성능 최적화 체크리스트

### 팁과 트릭 (Tips & Tricks)
- `performance_tips.md`: 성능 향상을 위한 실무 팁
- `debugging_guide.md`: 비동기 코드 디버깅 가이드
- `common_pitfalls.md`: 흔한 실수와 해결 방법

### 참고 자료 (References)
- `useful_links.md`: 도움되는 온라인 자료 링크
- `recommended_books.md`: 추천 도서 목록
- `glossary.md`: 용어집 (한영 대조)

### 코드 스니펫 (Code Snippets)
- `code_templates.py`: 자주 쓰이는 코드 템플릿
- `utility_functions.py`: 유용한 헬퍼 함수들

## 💡 활용 방법

### 학습 중
- 새로운 개념 학습 시 용어집 참조
- 실습 중 막힐 때 치트시트 확인
- 코드 작성 시 템플릿 활용

### 복습 시
- 성능 최적화 체크리스트로 점검
- 흔한 실수 가이드로 코드 리뷰
- 패턴 모음으로 설계 검증

### 실무 적용 시
- 유용한 링크로 최신 정보 확인
- 코드 스니펫으로 빠른 구현
- 디버깅 가이드로 문제 해결
