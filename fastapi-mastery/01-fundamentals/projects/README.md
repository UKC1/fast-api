# 01-Fundamentals 미니 프로젝트

## 🎯 프로젝트 목적
이론과 실습에서 배운 비동기 프로그래밍 개념을 종합하여 실제 동작하는 애플리케이션을 구축합니다.

## 📋 프로젝트 목록

### 1. Custom ASGI Server
**목표**: ASGI 서버를 처음부터 구현하여 FastAPI 내부 동작 완전 이해

**구현 계획**:
- 기본 HTTP 요청/응답 처리
- 미들웨어 스택 구현
- WebSocket 지원
- 정적 파일 서빙
- 기본적인 라우팅 시스템

**예상 소요 시간**: 4-6시간
**난이도**: ⭐⭐⭐⭐

### 2. Async Web Crawler
**목표**: 대규모 웹 크롤링으로 동시성과 리소스 관리 마스터

**구현 계획**:
- 동시 요청 수 제어 (세마포어)
- robots.txt 준수
- 요청 간 지연 설정
- 크롤링 결과 데이터베이스 저장
- 진행 상황 모니터링

**예상 소요 시간**: 3-4시간
**난이도**: ⭐⭐⭐

### 3. WebSocket Chat Server
**목표**: 실시간 통신으로 이벤트 기반 프로그래밍 체험

**구현 계획**:
- 다중 채팅방 관리
- 사용자 연결/해제 처리
- 메시지 브로드캐스트
- 채팅 이력 저장
- 간단한 웹 클라이언트

**예상 소요 시간**: 2-3시간
**난이도**: ⭐⭐⭐

## 📂 디렉토리 구조

```
projects/
├── custom-asgi-server/
│   ├── server.py           # 메인 ASGI 서버
│   ├── middleware.py       # 미들웨어 구현
│   ├── routing.py          # 라우팅 시스템
│   ├── websocket.py        # WebSocket 처리
│   ├── static/            # 정적 파일 테스트
│   └── tests/             # 서버 테스트
│
├── async-web-crawler/
│   ├── crawler.py          # 메인 크롤러
│   ├── robots_parser.py    # robots.txt 처리
│   ├── storage.py          # 데이터 저장
│   ├── config.py           # 설정 관리
│   └── examples/           # 크롤링 예제
│
└── chat-server/
    ├── server.py           # 채팅 서버
    ├── room_manager.py     # 채팅방 관리
    ├── client.html         # 웹 클라이언트
    ├── message_store.py    # 메시지 저장
    └── tests/             # 채팅 테스트
```

## 🎓 학습 효과

### Custom ASGI Server를 통해 배우는 것
- HTTP 프로토콜 저수준 처리
- 비동기 서버 아키텍처
- 미들웨어 패턴 구현
- 연결 관리와 리소스 정리

### Web Crawler를 통해 배우는 것
- 대규모 동시성 제어
- 외부 API/웹사이트 예의 바른 접근
- 데이터 파이프라인 구축
- 에러 핸들링과 재시도 로직

### Chat Server를 통해 배우는 것
- WebSocket 실시간 통신
- 이벤트 기반 아키텍처
- 다중 클라이언트 관리
- 상태 동기화 문제 해결

## 🚀 프로젝트 진행 순서

### 권장 순서 (초보자)
1. **Chat Server** → 가장 직관적이고 결과가 눈에 보임
2. **Web Crawler** → 동시성 개념 실무 적용
3. **ASGI Server** → 가장 복잡하지만 깊은 이해 가능

### 권장 순서 (경험자)
1. **ASGI Server** → 근본 원리부터 파악
2. **Web Crawler** → 실무 패턴 적용
3. **Chat Server** → 실시간 통신 마스터

## 💡 프로젝트별 핵심 학습 포인트

| 프로젝트 | 핵심 개념 | 실무 적용 |
|---------|----------|----------|
| ASGI Server | 서버 내부 구조, 미들웨어 | FastAPI 커스터마이징 |
| Web Crawler | 동시성 제어, 예의 바른 크롤링 | 데이터 수집 파이프라인 |
| Chat Server | 실시간 통신, 이벤트 관리 | 실시간 알림, 대시보드 |
