# 비동기 컨텍스트 매니저 실습

## 🎯 학습 목표
- 비동기 컨텍스트 매니저(`async with`) 완전 이해
- 리소스 자동 관리와 정리 패턴 마스터
- 데이터베이스 연결, 파일 I/O 최적화 실습

## 📁 예정 파일들
- `01_basic_context_manager.py`: 기본 async context manager 구현
- `02_database_connection.py`: DB 연결 관리 패턴
- `03_file_operations.py`: 비동기 파일 I/O 최적화
- `04_resource_pooling.py`: 리소스 풀 관리
- `05_error_handling.py`: 에러 상황에서 리소스 정리

## 🔧 실습 내용 미리보기
1. **기본 패턴**: `__aenter__`와 `__aexit__` 구현
2. **리소스 관리**: 자동 획득과 해제
3. **에러 핸들링**: 예외 발생 시 안전한 정리
4. **성능 최적화**: 연결 풀과 재사용 패턴
5. **실무 적용**: FastAPI에서 의존성 주입과 연계

## ⏰ 예상 학습 시간
- 기본 학습: 2-3시간
- 심화 실습: 3-4시간
- 프로젝트 적용: 1-2시간
