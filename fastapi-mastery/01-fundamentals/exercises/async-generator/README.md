# 비동기 제너레이터 실습

## 🎯 학습 목표
- 비동기 제너레이터(`async def` + `yield`) 마스터
- 대용량 데이터 스트리밍 처리 최적화
- 메모리 효율적인 데이터 파이프라인 구축

## 📁 예정 파일들
- `01_basic_async_generator.py`: 기본 async generator 구현
- `02_data_streaming.py`: 대용량 데이터 스트리밍
- `03_pipeline_processing.py`: 데이터 파이프라인 구축
- `04_infinite_sequences.py`: 무한 시퀀스 처리
- `05_fastapi_streaming.py`: FastAPI 스트리밍 응답

## 🔧 실습 내용 미리보기
1. **기본 구문**: `async for`와 `yield` 활용
2. **스트리밍**: 실시간 데이터 처리
3. **메모리 최적화**: 대용량 파일 처리
4. **파이프라인**: 여러 단계의 데이터 변환
5. **FastAPI 연동**: 스트리밍 응답 구현

## ⏰ 예상 학습 시간
- 기본 학습: 1-2시간
- 심화 실습: 2-3시간
- FastAPI 연동: 1-2시간
