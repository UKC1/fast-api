# 동시 HTTP 요청 처리 실습

## 🎯 학습 목표
- 여러 HTTP 요청을 효율적으로 동시 처리
- 타임아웃, 재시도, 서킷 브레이커 패턴 구현
- 외부 API 연동 시 안정성과 성능 보장

## 📁 예정 파일들
- `01_basic_concurrent_requests.py`: 기본 동시 요청
- `02_timeout_and_retry.py`: 타임아웃과 재시도 로직
- `03_circuit_breaker.py`: 서킷 브레이커 패턴
- `04_rate_limiting.py`: 요청 속도 제한
- `05_batch_processing.py`: 배치 요청 처리

## 🔧 실습 내용 미리보기
1. **동시성**: `asyncio.gather()`와 `aiohttp` 활용
2. **안정성**: 타임아웃과 예외 처리
3. **복원력**: 실패 시 재시도 로직
4. **보호**: 서킷 브레이커로 연쇄 장애 방지
5. **최적화**: 요청 배칭과 속도 제어

## ⏰ 예상 학습 시간
- 기본 학습: 2-3시간
- 패턴 구현: 3-4시간
- 실무 적용: 1-2시간

> **상태**: 🚧 준비 중 - 아직 구현되지 않음
